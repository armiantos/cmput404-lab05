# CMPUT404 - Lab05

Static site generators 🎉

## Requirements

- Make (3.81)
- Python (3.9.10)

## Getting started

1. Install dependencies

   ```sh
   pip install -r requirements.txt
   ```

2. Run dev server

   ```sh
   make devserver
   ```

## Publishing

To publish to github pages, run

```sh
make github
```
